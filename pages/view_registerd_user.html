🔄 View All Registered Users (GET Endpoint)

Let’s now learn how to create a GET API endpoint to retrieve all registered users from your MySQL database.

📁 File Structure (Where to Add)

project-root/
├── backend/
│   ├── server.js ✅ ← Add your new GET /users endpoint here
│   └── db.js

We’re updating server.js only. Nothing new needs to be installed.

⚙️ Step: Add GET /users Endpoint

Below your existing login route in server.js, add this:

app.get('/users', (req, res) => {
  const sql = 'SELECT id, name, email FROM users';
  db.query(sql, (err, results) => {
    if (err) return res.status(500).json({ error: 'Something went wrong.' });
    res.json(results);
  });
});

🧠 This selects all users from MySQL and returns them as JSON — perfect for an admin dashboard or mobile app.

📬 Test in Postman

  • ✅ Open Postman
  • 🧪 Method: GET
  • 🔗 URL: http://localhost:5000/users
  • 🧠 You should see a JSON array of all registered users

No token required (yet). This is a public endpoint for now. Later, we’ll add admin protection.

🧠 Final Notes (Layman Style)

  • 📥 db.query() talks to your MySQL database
  • 📦 You SELECT only name and email to avoid exposing passwords
  • 🧾 You return JSON so other apps (React, mobile) can use the data easily
