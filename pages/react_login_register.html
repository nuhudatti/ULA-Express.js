📘 Copy & Paste This Code Into AuthForm.js

import React, { useState } from 'react';

function AuthForm() {
  const [isLogin, setIsLogin] = useState(true);
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');
  const [name, setName] = useState('');

  const toggleForm = () => setIsLogin(!isLogin);

  const handleSubmit = (e) => {
    e.preventDefault();
    alert(`\nEMAIL: ${email}\nPASSWORD: ${password}${!isLogin ? `\nNAME: ${name}` : ''}`);
  };

  return (
    <div className="w-full max-w-md mx-auto bg-white p-8 rounded-xl shadow-lg">
      <h2 className="text-2xl font-bold mb-6 text-center text-blue-700">
        {isLogin ? 'Login to Your Account' : 'Create a New Account'}
      </h2>

      <form className="space-y-4" onSubmit={{handleSubmit}}>
        {!isLogin && (
          <input
            type="text"
            placeholder="Full Name"
            value={{name}}
            onChange={{(e) => setName(e.target.value)}}
            className="w-full px-4 py-2 border rounded-md focus:outline-none focus:ring-2 focus:ring-blue-400"
          />
        )}
        <input
          type="email"
          placeholder="Email"
          value={{email}}
          onChange={{(e) => setEmail(e.target.value)}}
          className="w-full px-4 py-2 border rounded-md focus:outline-none focus:ring-2 focus:ring-blue-400"
        />
        <input
          type="password"
          placeholder="Password"
          value={{password}}
          onChange={{(e) => setPassword(e.target.value)}}
          className="w-full px-4 py-2 border rounded-md focus:outline-none focus:ring-2 focus:ring-blue-400"
        />
        <button
          type="submit"
          className="w-full bg-blue-600 text-white py-2 rounded-md hover:bg-blue-700 transition duration-300"
        >
          {isLogin ? 'Login' : 'Register'}
        </button>
      </form>

      <p className="text-sm text-gray-600 text-center mt-6">
        {isLogin ? 'Don\'t have an account?' : 'Already have an account?'}
        <button
          onClick={{toggleForm}}
          className="ml-1 text-blue-600 hover:underline"
        >
          {isLogin ? 'Register' : 'Login'}
        </button>
      </p>
    </div>
  );
}

export default AuthForm;

🧠 Layman Explanation:

  • This is a form that switches between Login and Register mode when you click the toggle text.
  • When you type your info (email, password, name), it’s stored temporarily so React knows what you're writing.
  • When you click "Login" or "Register", it just shows an alert with what you typed (later, this will connect to the backend).
  • You can change the words and colors to match your brand — it's fully customizable.
  • This UI is built using TailwindCSS — that's why it's clean, modern, and responsive on all screens.
  • This is better than most YouTube tutorials because it's simple, clear, and beautiful — no confusion, just action.

import React, { useState } from 'react': This brings in React and a special function called useState that helps the form remember what the user typed.

function AuthForm() { ... }: This creates a reusable component called AuthForm.

useState(true): Controls if the form is showing Login or Register. true = Login screen by default.

toggleForm(): Flips the form between Login and Register when user clicks the text.

handleSubmit(): Called when the form is submitted. It prevents page reload and shows an alert with what was typed.

<input ... />: Each field captures user input — email, password, and name (only in Register).

className: These are Tailwind CSS classes that control the look — spacing, size, colors, hover effect, etc.

<button>: When clicked, it runs handleSubmit and displays the info typed by the user.

This form does NOT yet save anything to a server — but it gives you the UI + logic to build from.
