Frontend Setup Guide

We'll set up the React application with Tailwind CSS for styling. This is like building the foundation of your house before adding walls and furniture. Don't worry - I'll explain each step in simple terms!

Node.js

JavaScript runtime environment

React

Frontend library for UI

Tailwind CSS

Utility-first CSS framework

What We're Creating

We're setting up a new React project called "portfolio-frontend" with Tailwind CSS for styling. This creates all the necessary files and folders to start building your portfolio website.

1

Install Node.js

Node.js is like the engine that runs JavaScript outside of the browser. We need it to create and run our React application.

Windows & macOS

  1. Visit the Node.js website
  2. Download the LTS version (Long Term Support)
  3. Run the installer with all default settings
  4. Restart your computer after installation

Verify Installation

node -v

Should display version v16.x.x or higher

npm -v

Should display version 8.x.x or higher

$ node -v
v18.12.1
$ npm -v
8.19.2
2

Create Your React App

We'll use Create React App - a tool that sets up a new React project with all necessary configurations. It's like a starter kit for React applications.

Run These Commands in Your Terminal

npx create-react-app portfolio-frontend

This creates a new React project called "portfolio-frontend". It might take 1-2 minutes.

cd portfolio-frontend

This moves you into the project directory.

What Just Happened?

Create React App generated all the files and folders needed for a basic React application. You now have a working starter project!

3

Install Dependencies

Dependencies are like tools in your toolbox. We need a few extra packages for routing and styling.

Install These Packages

npm install react-router-dom axios
react-router-dom: For navigating between pages
npm install -D tailwindcss postcss autoprefixer
Tailwind CSS: For styling our components
npx tailwindcss init -p

This creates the configuration files for Tailwind CSS.

4

Configure Tailwind CSS

We need to set up Tailwind CSS so it knows which files to look at and which styles to generate.

Update Tailwind Configuration

tailwind.config.js
module.exports = {
  content: [
    "./src/**/*.{js,jsx,ts,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

This tells Tailwind to look for classes in all JavaScript/JSX files in your src folder.

Add Tailwind Directives to CSS

src/index.css
@tailwind base;
@tailwind components;
@tailwind utilities;

This imports Tailwind's styles into your application.

5

Start Development Server

Let's start the development server to see our React application in action!

npm start

This command will automatically open your default browser to http://localhost:3000

Congratulations!

You've successfully set up your React project with Tailwind CSS. You should see the default React welcome page. We'll replace this with our portfolio in the next steps.

Setup Complete!

Your development environment is ready
