CORS Configuration

Cross-Origin Resource Sharing (CORS) must be configured on your backend to allow requests from your frontend domain.

server.js (Backend)
1
const express = require('express');
2
const cors = require('cors');
3
4
const app = express();
5
6
// Configure CORS
7
const corsOptions = {
8
  origin: 'http://localhost:3000', // Your frontend URL
9
  optionsSuccessStatus: 200 // Legacy browser support
10
};
11
12
app.use(cors(corsOptions));

CORS Note

For production, configure allowed origins properly. Never use app.use(cors()) without options in production as it allows all domains.
