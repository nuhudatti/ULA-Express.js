Why Use Axios?

Axios provides several advantages over the native Fetch API:

  • Automatic JSON data transformation
  • Better error handling
  • Request and response interception
  • Built-in XSRF protection
  • Progress tracking for uploads

CORS Configuration

Cross-Origin Resource Sharing (CORS) is a security feature that restricts cross-origin HTTP requests. Since our frontend and backend run on different ports during development, we need to configure CORS properly.

Frontend (Port 3000)

Sends request to backend

Backend (Port 5000)

Must include CORS headers in response

Environment Variables

For security and flexibility, we should move configuration like API base URLs into environment variables.

Frontend (.env)

.env
REACT_APP_API_BASE_URL=http://localhost:5000
REACT_APP_ENV=development

Prefix with REACT_APP_ to make them available in React

Backend (.env)

.env
PORT=5000
NODE_ENV=development
ALLOWED_ORIGINS=http://localhost:3000,https://yourdomain.com

Require dotenv package in your server.js

src/services/api.js

// Updated API configuration

Security Note

Never commit .env files to version control. Add them to your .gitignore file:

.gitignore
# Environment variables
.env
.env.local
.env*.local
